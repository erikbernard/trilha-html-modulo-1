Estrutura Básica do HTML

(voltar)

Páginas

Todo site é formado por uma estrutura HTML onde cada que são agrupadas por páginas, os modelos de SPA contêm apenas uma únicas paginas(arquivo index). E a primeira página é chamada de "Index"(índice). Note que isso não é uma regra, mas é uma convenção da comunidade. Então o emso pode ser alterado.

Dentro da Tag HTML

<html>

Todo documento HTML inicia como essa declaração <!DOCTYPE html>. posteriomente e aberta a tag <html>, onde irá contêm nossas demais estrututas.

<head>

A tag head faz parte da estrutura básica do documento, sua finalidade é definir o cabeçalho do documento com informações que não serão exibidas dentro do conteúdo da página.

<title>

Os títulos são muito importantes em SEO (Search Engine Optimization). Ou seja, se quiser que seus sites sejam bem rankeados pelos resultados de busca, você deve dar um título para todas as páginas de seu site.

<meta>

Define qualquer informação de metadados que não podem ser definidos por outros elementos HTML. Meta tags são extremamente importantes do ponto de vista de SEO, pois permitem que os mecanismos de pesquisa categorizem o conteúdo do site corretamente.

<body>

Literalmete o corpo onde ira conter o que vai aparecer para o usuário. que junto com a tag <html> e a <head> formam a tríade de tags mais importantes e essenciais de qualquer site feito em HTML.

Falando Sobre Tags

(voltar)

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing) como <img>, <br>, etc.

O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Existem muitas tags, e cada uma tem uma função.Por exemplo:

Tag <i>

A <i> define uma parte do texto em uma voz ou humor alternativo. O conteúdo dentro é normalmente exibido em itálico .

A <i> é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.

Tag <strong>

A tag <strong> usada para definir o texto com forte importância. O conteúdo dentro é normalmente exibido em negrito.

Atributo das Tags

(voltar)

Atributos são propriedades que uma tag pode ter. Existem atributos globais e atributos específicos.

Atributos Globais

Atributos globais se aplicam a todos os elementos HTML, pois eles fornecem informações adicionais que os navegadores usam para determinar a forma como os elementos são exibidos na página.

Atributo id

O atributo global id define um identificador exclusivo (ID) que deve ser único por todo o documento. Seu objetivo é identificar o elemento ao navegar por âncoras (usando um identificador de fragmento), quando utilizar scripts ou estilizando (com CSS).

Atributo style

O atributo global style contém declarações de estilo CSS a serem aplicadas ao elemento. Observe que é recomendado que os estilos sejam definidos em um arquivo ou arquivos separados. Este atributo e o elemento têm principalmente a finalidade de permitir um estilo rápido, diretamente no elemento, por exemplo, para fins de teste.

Atributo Class

As classes permitem que CSS e JavaScript selecionem e acessem elementos específicos por meio de seletores de classe ou funções como o método DOM .class document.getElementsByClassName

Atributos específicos

São atributos que apenas uma ou algumas tags possuem acesso, são atributos próprios delas. Exemplo:

Atributo type

É específico da tag <input>. Muda o tipo de campo que será usado no input.

Atributo src

Específico da tag <img>. Define o endereço da imagem que vai aparecer.

Entender de tipografia * é de suma importância para o desenvolvedor Web

Um dos principais objetivos do HTML é dar estrutura de texto e significado, também conhecido como semântica), para que um navegador possa exibi-lo corretamente.

Títulos

São as tags <h1>, <h2>, <h3>, <h4>, <h5> e <h6>.

Existem níveis como título, subtítulo etc.

  • <h1>

  • <h2>

  • <h3>

  • <h5>
  • <h6>

Tag <h1>

É o título principal da página. Só deve ter uma tag <h1> por página. O google penaliza sites que contém mais de um <h1> por página.

Parágrafos

São definidos pela tag <p>. A cada novo parágrafo, se coloca uma tag <p>

Citação

Para fazer uma citação,ou uma observação se usa a tag <blockquote>. Exemplo:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Inventore harum aspernatur cumque debitis, nisi incidunt amet, temporibus commodi in et molestias? Hic aut vero quae. Debitis laboriosam cupiditate mollitia maxime.

Note que foi dado um espaçamento, que também pode ser alterado via CSS.

Margens

O navegador, por padrão, dá um espaçamento entre títulos e Parágrafos. Mas isso pode ser alterado no CSS.

Você pode conferir essas margens usando a Ferramenta do Desenvolvedor do seu navegador.

Listas Ordenadas e não Ordenadas

(voltar)

Listas são muito usadas no desenvolvimento Web. As listas HTML permitem que os desenvolvedores da Web agrupem um conjunto de itens relacionados em listas. As principais do HTML SÃO:

Listas Ordenadas

Uma lista ordenada começa com a tag <ol> (ordered list). Cada item da lista começa com a tag filha * <li> (list item). Obs: a tag <ol> não funciona se não colocar a tag <li>.

Os itens da lista serão marcados com números por padrão:

  1. Item 1
  2. Item 2
  3. Item 3

Pode se colocar uma lista dentro da outra:

  1. Item 1
    1. item 1.1
    2. item 1.2
    3. item 1.3

Listas Não Ordenadas

Uma lista não ordenada começa com a tag <ul> (unordered list). Cada item da lista começa com a tag filha <li> (list item). Obs: a tag <ul> não funciona se não colocar a tag <li>.

Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão:

  • Item 1
  • Item 2
  • Item 3
obs: As listas podem ser usadas para fazer um sumário, mas via CSS pode se usar para menus.

Links

(voltar)

Em todo site, você clica em um link *, e vai para outra página. Isso é o que chamamos de "navegar na internet".

No HTML, a navegação é feita através da tag <a>(âncora)*.

A tag <a> tem que vir com o atributo href, que indica o destino do link.Exemplo:

<a href="https://www.dio.me/">site da dio</a>

Para abrir em uma nova aba, se coloca o atributo target="_blank". Ex:

<a href="https://www.dio.me/" target="_blank">site da dio</a>

Uma forma alternativa para abrir links em uma nova aba é utilizar as teclas de atalho CTRL+click.
